Adds a customizable 5 star rating field to the worpress native comment form.. 

== Description ==
MM Comment Field Ratings adds a 5 star rating field to the comment form in WordPress, allowing the site visitor to optionally submit a rating along with their comment. Ratings are displayed as stars below the comment text.

= Standard features =
1. Six styles you can use to change the look.
2. You can also customize the colors as you wish.
3. Add mutliple comment field and customize them individually
4. You can make it as required field, change required message.
5. Enable or Disable average rating to show in comments.
6. Add your custom css.

== Installation ==
* Download the plugin and extract it.
* Upload the directory /mm-comment-field-ratings/ to the /wp-content/plugins/ directory.
* Activate the plugin through the \'Plugins\' menu in WordPress.
* Click on 'MM Comment Ratings' from the left menu bar.
